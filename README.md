## Polyglot

Polyglot is a simple internationalization-first content management system built
on top of Firebase.

[Demo](https://polyglottest-5124c.firebaseapp.com/test.html)

Polyglot is released under [WTFPLv2](http://www.wtfpl.net/about/).
